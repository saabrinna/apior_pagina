Hamburgueria Delícia Suprema 3000

Este projeto foi desenvolvido como atividade das aulas de Design Web e Programação Web, com o objetivo de criar propositalmente o pior layout possível de um site de uma hamburgueria.

Integrantes do grupo:
Ana Laura Matos
Sabrina Cardoso
Yasmim Silva

Objetivo do projeto:
A proposta da atividade foi aplicar os conhecimentos de HTML, CSS e JavaScript, mas de forma a quebrar as boas práticas de design, UI e UX, criando um site com a pior experiência possível para o usuário.
O intuito era compreender na prática quais erros devem ser evitados no desenvolvimento de páginas web.

O que foi feito no site:
No site foram criadas várias páginas interligadas, todas com estilos e comportamentos intencionalmente ruins, incluindo:

Página inicial (index.html): contém o título do site, um menu de navegação e mensagens animadas piscando.
Página Home: apresenta uma mensagem de boas-vindas com texto em movimento e cores conflitantes.
Página Cardápio: mostra uma lista de hambúrgueres com preços exagerados e um layout desorganizado.
Página Contato: possui um formulário confuso, com cores fortes e campos desalinhados.
Página Promoção Secreta: exibe um alerta automático e um “cupom” falso, com cores chamativas e elementos piscando.
Além disso, o site utiliza JavaScript para gerar alertas constantes e efeitos irritantes, tornando a navegação propositalmente incômoda.

Tecnologias utilizadas:
HTML5
CSS3
JavaScript
